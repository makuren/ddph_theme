# ddph_theme
Wordpress Theme for Daredemo PH website
For collaborators only:

Requirements:
- MAMP/XAMPP for easy running

1) Copy wp-config.php template from boilerplate wordpress installation and configure.
2) Run server.
3) Go to Appearance > Themes and change theme to DDPH Zero.
4) Perform changes.
5) Can push directly for collaborators, PR for non-collaborators.
